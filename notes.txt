Git now uses main as its default branch
GIT STEPS
1. move to project  folder
2. git init
3. git add .
4. git commit -m "Initial commit"
5. git remote add origin https://github.com/username/repository.git
6. git push -u origin master (if you are in master branch)



To delete commits
1. git checkout (branchname)
2. git rebase -i HEAD~5
3. edit the file (pick -> drop to delte, edit to change it)
    vim ( :wq! changes saved       :qa! exit without saving)
4. git rebase --continue
5. git push --force
